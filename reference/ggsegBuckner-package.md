# ggsegBuckner: Buckner Cerebellar Functional Parcellation for the 'ggseg' Ecosystem

Buckner cerebellar parcellation into 7 or 17 functional networks for the
'ggseg' ecosystem. Provides a unified 'ggseg_atlas' object with both 2D
polygon geometry and 3D vertex indices, for use with 'ggseg' and
'ggseg3d'.
